\begin{problem}[0.15]
  Show that $S = \{x \in \R \mid 0 < x < 1\}$ has the same cardinality as $\R$. [Hint: Find an elementary function of calculus that maps an interval one-to-one onto $\R$, and then translate and scale appropriately to make the domain the set $S$.]
\end{problem}

\begin{solution}
  Take the function $f(x) = \tan(\pi x - \pi/2)$. This function maps the interval $(0, 1)$ one-to-one onto $\R$. Specifically, as $x$ approaches $0$ from the right, $f(x)$ approaches $-\infty$, and as $x$ approaches $1$ from the left, $f(x)$ approaches $\infty$. The function is continuous and strictly increasing on the interval $(0, 1)$, ensuring that it is a bijection between the set $S = \{x \in \R \mid 0 < x < 1\}$ and $\R$. Therefore, $S$ has the same cardinality as $\R$.
\end{solution}
